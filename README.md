# SimpileCompiler

SimpileCompiler
编译器

## 使用方法

```shell
git clone https://github.com/Simple-XX/SimpleCompiler.git
cd SimpleCompiler
mkdir build
cd build
cmake ..
./bin/SimpleCompiler -h
```
